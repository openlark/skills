## Universal Compatibility Paths

Most widely adopted cross-product skill sharing paths:

- `~/.agents/skills/` — user-level
- `.agents/skills/` — project-level

Some implementations also scan `.claude/skills/`, `.codex/skills/` etc. Lenient validation ensures cross-product skill interchangeability.
