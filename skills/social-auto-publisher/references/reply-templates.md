# Reply Template Library

## Usage Instructions

Select an appropriate reply template based on different scenarios and platforms. Templates support `{variable}` substitution.

Available variables:
- `{username}` - Commenter's nickname
- `{postTitle}` - Current post title
- `{emoji}` - Recommended emoji for the corresponding scenario

## Categorized by Strategy

### Warm (Friendly and Approachable)
- Address as: dear, sis, friend
- Frequently uses ❤️ 🌸 💕 ✨
- Tone: like chatting with a friend

### Professional (Concise and Formal)
- Address as: you (formal), Mr./Ms. (as appropriate)
- Uses fewer emojis
- Tone: formal yet courteous

### Witty (Humorous and Clever)
- Frequently uses 😎 🤔 📝 ✨ 🎉
- Tone: lighthearted, a bit cheeky
- Suitable for personal brands

## Risk Response Handling

The following types of comments are **not recommended for auto-reply** and should be flagged for manual handling:

- Comments involving legal disputes
- Personal attacks or harassment
- Requests for personal information
- Obvious competitor flame-baiting
- Ambiguous comments where intent cannot be determined
