# Content Strategy and Topic Selection Methodology

## Content Pyramid

```
        ┌─────────┐
        │  Viral  │  10% - Viral dissemination level
        │   ───   │
        │ Quality │  30% - Consistently high engagement
        │  ─────  │
        │ Regular │  60% - Daily activity maintenance
        └─────────┘
```

## Topic Sources

### 1. Trend Tracking
- Weibo trending topics
- Xiaohongshu trending topics
- Twitter trending
- Google Trends
- Industry news / Product launches

### 2. User Demand
- Follow-up questions and feedback in the comments section
- High-frequency questions in direct messages
- Unmet needs found in competitor comment sections

### 3. Content Matrix
- **Tutorials**: How-to articles, tutorials, guides
- **Opinions**: Insights, analysis, predictions
- **Stories**: Experiences, case studies, retrospectives
- **Interactive**: Polls, questions, challenges
- **Resources**: Tool recommendations, book lists, collections

### 4. Competitor Analysis
- Follow viral content in the same niche
- Analyze why it went viral (Headline? Topic selection? Timeliness?)
- Differentiate your approach (Different angle / Deeper dive / More practical)

## Topic Evaluation Quadrant

```
        High Value
            │
    Ⅱ       │      Ⅰ
  Benchmark │  Key Investment
  ──────────┼──────────
  Low Heat  │   High Heat
    Ⅲ       │      Ⅳ
   Hold     │  Quick Output
            │
        Low Value
```

- **Ⅰ High Value + High Heat**: Key investment; aim for viral content
- **Ⅱ High Value + Low Heat**: Build benchmark content; cultivate the long tail
- **Ⅲ Low Value + Low Heat**: Hold or do not publish
- **Ⅳ Low Value + High Heat**: Quick output; ride the traffic wave

## Content Preferences by Platform

| Dimension | Xiaohongshu | Weibo | Twitter |
|-----------|-------------|-------|---------|
| Title Style | Numbers + Emoji + Keywords | Hashtags + Concise | Hook + Suspense |
| Content Depth | In-depth + Practical | Concise + Timely | Insight + Opinion |
| Image Needs | Aesthetic, lifestyle feel | Infographics, screenshots | Clean, data-driven |
| Engagement Method | Saves > Likes > Comments | Retweets > Comments > Likes | Replies > RT > Likes |
| Viral Cycle | 3-7 days | 1-2 days | 1-3 days |

## Data Review Framework

Review each piece of content 72 hours after publishing:

1. **Impressions** — Did they meet expectations?
2. **Click-Through Rate** — Is the title/cover attractive?
3. **Engagement Rate** — Did the content spark discussion?
4. **Conversion** — Follower growth / Saves / Retweets?
5. **Improvement Points** — How can the next post be optimized?
