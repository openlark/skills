# Market Analysis Report Writing

Write structured, insightful market analysis reports as a professional market analyst. Use business-professional language to provide valuable references for decision-makers.

## Use Cases

Use when the user needs "market analysis report", "industry analysis report", "market research report", "competitive analysis report", or "industry research".

## Report Structure

Organize reports according to the following structure to ensure comprehensive coverage:

1. **Industry Overview** — Industry definition, key characteristics, major participants, key drivers, development history and current status
2. **Market Status Analysis** — Market size, growth rate, market share, key trends and dynamics
3. **Future Development Trends** — Macro environment (PESTEL), technological advances, policy changes, potential opportunities
4. **Competitive Landscape** — Major competitors, market share, strategic advantages, response strategies
5. **Consumer Behavior** — Target consumer profiles, needs and purchasing behavior, marketing strategy recommendations
6. **Conclusions and Recommendations** — Summary of key findings, forward-looking recommendations

## Workflow

### 1. Information Gathering

Use `web_fetch` and search tools to obtain data from reliable sources:

- Prioritize authoritative sources: government statistics, industry association reports, well-known consulting firms (McKinsey, BCG, Deloitte, etc.), public company financial reports
- Cross-validate key data to avoid single-source dependency
- Cite data sources and dates

### 2. Report Writing

- Use business-professional language, avoid colloquial expressions
- Cite data sources and years for all data references
- Provide in-depth analysis — not just describing phenomena, but explaining causes and impacts
- Maintain neutrality and objectivity, avoid personal opinions or bias
- Conclusions and recommendations must be based on preceding analysis and be actionable

### 3. Output Format

Reports use Markdown format with clear hierarchy:

- Use `#` `##` `###` to organize levels
- Present key data in tables
- Use lists to organize key points
- Include data sources at the end of the report

## Notes

- Output only the report body, without any descriptive commentary
- All data must be based on reliable sources; unverified information should not be used
- Reports should demonstrate market insight, not mere fact listing
- Flexibly adjust chapter weights based on industry characteristics; do not apply mechanically
