# Chapter Writing Workflow

## Pre-Write Checklist

Complete in order, produce the **chapter writing brief**:

### 1. Context Review

```
- Previous chapter's ending scene:
- Previous chapter's unresolved conflict/suspense:
- Previous chapter's reader emotional landing (tense/relaxed/anticipating/sad):
- This chapter's starting scene:
```

### 2. Foreshadowing Status

```
【Planned operations for this chapter】
- Foreshadowing to resolve (ID + content):
- New foreshadowing to plant (content + planned resolution chapter):
- Foreshadowing to advance but not resolve (ID + degree of advancement):
```

### 3. Character Status

```
【Characters appearing in this chapter】
- Character A: Current location / current emotion / this chapter's goal / this chapter's function
- Character B: ...
【Characters not appearing but need to remember their status】
- Character C: Currently in XX state at XX location
```

### 4. Pacing Position

```
【Chapter type】 (Climax / Transition / Setup / Twist)
【Position within volume】 (First third / Middle / Closing)
【Emotional design】 (Opening emotion → Process emotion → Ending emotion)
【Information density】 (High / Medium / Low) — how much info reader needs to digest
```

## During Writing

- **First 300 words decide life or death**: must have conflict/suspense/emotion (at least one)
- **Every 2000 words a mini-rhythm**: info release → minor conflict → minor resolution
- **Dialogue must have "subtext"**: what character says ≠ everything they're thinking
- **Scene transitions need "hooks"**: don't let readers put it down when switching scenes
- **Battle/climax scenes**: faster rhythm, shorter sentences

## Post-Write Review

### Basic Checks

- [ ] Did this chapter complete the rhythm beat structure (Hook→Setup→Development→Twist→Climax→Cliffhanger)?
- [ ] Word count within expected range (deviation <20%)?
- [ ] Chapter-ending cliffhanger effective (will readers want to click next)?

### Consistency Checks

- [ ] Character dialogue and behavior match their profile?
- [ ] Any content conflicting with established settings?
- [ ] Timeline coherent (reasonable gap from previous chapter)?
- [ ] Character abilities/levels consistent with setting?

### Foreshadowing Checks

- [ ] New foreshadowing planted this chapter logged?
- [ ] Planned resolutions correctly resolved? Resolution method natural?
- [ ] Any "forgotten" foreshadowing needing advancement?

### Style Checks

- [ ] Dialogue style matches characters?
- [ ] Narrative rhythm matches chapter type? (Climax fast, Setup slow)
- [ ] Emotional tone matches current volume's setting?

## Common Issues & Fixes

| Issue | Diagnosis | Fix |
|-------|-----------|-----|
| Chapter too flat | No hook/twist/climax | Rewrite opening hook, add at least one minor reversal |
| Padding | Info density too low, plot not advancing | Cut excess description/psychology, add event progression |
| Setting conflict | Character ability or rules inconsistent | Check setting profile, fix chapter or update setting |
| Character drift | Personality inconsistent with earlier | Check character profile, rewrite relevant dialogue/behavior |
| Homogenized dialogue | All characters talk the same | Design 1-2 linguistic quirks per character |
| Broken rhythm | Abrupt scene transitions | Add transition sentences / emotional bridges |
| Weak hook | No suspense or anticipation at chapter end | Leave an unanswered question or an impending conflict |
