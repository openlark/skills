# Worldbuilding & Character Setting Template

## I. World Foundation

```
【Era/World Type】
- Era: Ancient / Modern / Future / Alternate
- Type: Xianxia / Fantasy / Urban / Sci-fi / Post-apocalyptic / Game / Historical
- Relation to reality: Parallel world / Historical branch / Pure alternate

【World Rules】
- Power system (cultivation levels / magic classification / tech tier)
- Special rules (what makes this world "different")
- Limitations and costs of the rules

【Geography & Factions】
- Key regions (map overview)
- Faction distribution (nations/sects/organizations, relationship map)
- Resource distribution (what is where, who controls it)
```

## II. Protagonist Profile

```
【Basic Info】
- Name/title, age, appearance (3 memorable traits)
- Personality keywords (≥3): e.g. "reserved, decisive, loyal"
- Personality paradox (adds depth): e.g. "cold exterior, passionate heart"

【Background】
- Origin: family, social class, upbringing environment
- Core trauma/drive: the fundamental force driving the protagonist
- Starting state: situation at the beginning of the story

【Growth Arc】
- Starting state → End state
- Key growth milestones (≥5)
- Cheat/special ability (type, limitations, cost)

【Relationship Network】
- Core relationship characters and relationship evolution directions
```

## III. Supporting Characters

Minimum for each major character:

```
【Name/Role】
【Appearance tag】 (1-2 traits for readers to remember)
【Personality tags】 (2-3 keywords)
【Relationship with protagonist】 and evolution curve
【Narrative function】 (advance plot / create conflict / provide info / emotional support)
【Independent motivation】 (character doesn't exist solely for the protagonist)
```

Character tiers:
- S-tier (throughout the entire book, 3-5 people) → full profile
- A-tier (one volume, 5-10 people) → core profile
- B-tier (one part, functional) → brief notes

## IV. Core Conflict System

```
【Main Conflict】
- Conflict nature: Person vs Person / Person vs Environment / Person vs Self / Person vs Fate
- Opposition: antagonist/hostile faction
- Conflict escalation path: Small → Medium → Large → Ultimate

【Phase Goal Breakdown】
- Volume 1: Achieve XXX
- Volume 2: Achieve XXX
- ...

【Antagonist/Obstacle Design】
- Main antagonist: motivation (must be reasonable), capability, relationship with protagonist
- Phase antagonists: opponents per volume/part
- Non-human obstacles: environment, systems, fate, etc.
```

## V. Setting Consistency Log

```
【Established Iron Laws】 (immutable)
- Rule 1: XXX (established in Ch. X)
- Rule 2: XXX

【Power System】
- Level names, each level's characteristics, advancement conditions
- Protagonist's current level, advancement progress

【Timeline】
- Story start timepoint
- Key event timepoints
```

## Usage Tips

- Complete items I-IV when starting a new book
- Update item V before each volume begins
- Update instantly when introducing new characters/settings
- When conflicts arise, prioritize "iron laws," adjust non-iron-law content
