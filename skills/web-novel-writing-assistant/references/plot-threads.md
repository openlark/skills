# Multi-Plot Management & Status Board

## Plotline Classification

```
【Main Plot】 — Protagonist's core objective line, runs through the entire book
【Major Subplots】 — Secondary lines affecting the main plot (2-4)
【Minor Subplots】 — Character growth / romance / worldbuilding lines (several)
【Hidden Plot】 — Hidden threads, revealed later
```

## Status Board Template

Update after every chapter written. Status: 🟢Active / 🟡Dormant / 🔴Resolved / ⚪Not Started

```

```

## Convergence Planning

When multiple lines need to intersect, plan intersection points:

```
【Convergence Point #1】 (Estimated Ch. X - Y)
Lines converging: Main + Subplot A + Subplot B
Convergence method:
- Location convergence: all characters arrive at same location
- Event convergence: one event simultaneously affects multiple lines
- Information convergence: characters on different lines discover pieces of the same info
Post-convergence direction:
- Merge into one line
- Continue separately but mutually influencing
- One line ends/goes dormant
```

## Priority & Tradeoffs

When "too many lines to write":

1. Main plot has highest priority, cannot go more than 3 chapters without advancing
2. Major subplots sorted by reader interest; low-interest ones can go dormant
3. Dormant lines can't "disappear" — need occasional mentions so readers remember
4. A line not advanced in >50 chapters: consider accelerating resolution or cutting
5. New line must be evaluated: are existing lines already too many?

## Rhythm Distribution Suggestions

Over a 100-chapter cycle:

```
Main plot: 60-70 chapters (maintain core momentum)
Major subplot A: 15-20 chapters (dedicated segments)
Major subplot B: 10-15 chapters
Minor subplots: interspersed, no dedicated chapters
Hidden plot: one clue fragment every 20-30 chapters
```

## Check Questions

Self-check every 30 chapters:
- Which line hasn't advanced in 20+ chapters?
- Which line do readers care about most in comments?
- Has any line "died naturally" (nobody cares)?
- Are intersection points for next phase clear?
