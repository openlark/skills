# Conflict Detection Matrix

## I. Setting Conflict

Run after writing each chapter:

### Ability/Level Consistency

```
- Character ability matches their current setting level?
  □ Protagonist's overlevel fights have reasonable explanation?
  □ Supporting character abilities consistent with their level?
- Cultivation/advancement rules consistent front to back?
  □ Same-level characters' power gaps reasonable?
```

### World Rule Consistency

```
- World rules "forgotten" or "violated"?
  □ Power system limitations broken (without explanation)?
  □ Special rules consistent across chapters?
  □ If rules are broken, is there foreshadowing and a cost?
```

### Timeline Consistency

```
- Time passage reasonable?
  □ Event intervals proportional to narrative length?
  □ Time contradictions between scenes?
  □ Character ages / seasons / holidays all check out?
```

## II. Character Conflict

### Personality Consistency

```
- Character decisions match their personality profile?
  □ Decisive characters becoming suddenly indecisive?
  □ Rational characters unreasonably dumbed down?
  □ Character emotional reactions match their personality?
```

### Relationship Consistency

```
- Character relationships suddenly changing (no transition)?
  □ Hostile→friendly: reasonable reversal process?
  □ Close→distant: foreshadowing event?
  □ New character interactions match both personalities?
```

### Motivation Consistency

```
- Character actions have clear motivation?
  □ Supporting characters' actions exist only to advance plot?
  □ Character goals contradictory front to back?
```

## III. Plot Logic

### Causal Chain

```
- "Because A, therefore B" logic chain complete?
  □ Any "somehow this happened" moments?
  □ Key reversals have sufficient foreshadowing?
  □ Too many coincidences? (≤1 coincidence per chapter is reasonable)
```

### Information Acquisition

```
- Character info-gathering methods reasonable?
  □ "Overheard the key conversation by coincidence"?
  □ Characters knowing information they "shouldn't know"?
  □ Info reveal rhythm reasonable (too fast or too slow)?
```

### Dumbing-Down Detection

```
- Any of these dumbing-down patterns?
  □ Protagonist not making the simplest reasonable choice at critical moments
  □ Antagonist not killing protagonist when they could (no reasonable explanation)
  □ Supporting cast collectively dumbed down to highlight protagonist
  □ Crisis resolved by "just so happening that..." (deus ex machina)
```

## IV. Numerical Consistency

```
- Cultivation levels: count/name/features recorded and consistent?
- Time data: spans, ages, event intervals self-consistent?
- Money/resources: economic system reasonable?
- Distance/scale: geography, army sizes front-to-back consistent?
```

## V. Conflict Resolution Principles

**When conflicts are found:**

1. **Published chapter settings are "iron law"** → fix unpublished current chapter to align
2. **If published chapter itself has a bug** → post correction notice or silently fix (per platform rules)
3. **If "setting upgrade" not contradiction** → ensure reasonable transition explanation
4. **Log all conflicts and resolutions** → prevent recurrence

## VI. Quick-Reference Mnemonic

```
Abilities check levels, time checks the line
Characters check personality, plot checks causality
Coincidences max three, dumbing-down is the worst crime
Numbers keep clear record, foreshadowing must close the loop
```
