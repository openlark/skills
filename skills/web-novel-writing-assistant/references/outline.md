# Three-Level Outline Template

## I. Complete Book Outline (Volume Level)

```
【Book Basic Info】
- Title, genre, estimated total word count
- Core selling point (one sentence: why readers should read this)
- Target reader profile

【Volume Structure】
Volume 1: "Volume Title" (~XX words, XX chapters)
- Volume theme: What this volume is about
- Opening hook: What grabs readers in the first 3 chapters
- Core conflict: Main tension of this volume
- Key events (≥5):
  1. XXX (approx chapter range)
  2. XXX
  ...
- Volume ending: What is achieved, what suspense is left

Volume 2: "Volume Title" ...
```

## II. Part Outline (Within Each Volume)

Break each volume into 3-5 parts, each with setup-development-turn-resolution:

```
【Part X】 (Ch. X - Ch. Y)
- Part theme:
- Setup (opening): How this story segment begins
- Development (progression): How the story advances
- Turn (twist): Key reversal or revelation
- Resolution (closure): How this part wraps up, setting up next
- Information release: What readers need to learn in this part
- Foreshadowing: What to plant, what to resolve
```

## III. Chapter Outline (Per-Chapter Rhythm Template)

```
【Chapter X: Chapter Title】

Positioning: Climax chapter / Transition chapter / Setup chapter / Twist chapter

Rhythm Beat Design:
1. Hook: What grabs the reader in the first paragraph/scene
2. Setup: Establish this chapter's situation, tell reader what's at stake
3. Development: Plot unfolds, information released
4. Twist: Surprise, conflict, revelation — the chapter's dramatic core
5. Climax: Peak or phased resolution of this chapter's conflict
6. Cliffhanger: Suspense, preview, emotional continuation

Involved plotlines: (Main / Subplot A / Subplot B)
Characters appearing:
New settings/foreshadowing:
Estimated word count:
```

## IV. Outline Flexibility Principles

- **Outline is a map, not railway tracks**: deviation allowed but must return to main line
- **Volume-level outline**: sets direction, not easily changed
- **Part-level outline**: adjustable, but assess impact on subsequent
- **Chapter-level outline**: must do before writing, can adjust during writing
- **Buffer chapters**: reserve 2-3 chapters per volume for insertion or rhythm adjustment

## V. Serialization Rhythm Suggestions

| Words/Day | Type | Strategy |
|-----------|------|----------|
| 4000+ | High-frequency serial | Detailed chapter outlines, backlog ≥30 chapters |
| 2000-4000 | Standard serial | Complete chapter outlines, backlog ≥15 chapters |
| <2000 | Low-frequency serial | Focus on per-chapter quality, outline can be rougher |
