### Step 2: Theoretical Support

**Core Theory**: **"Conflict Opening" Principle** (from Robert McKee's *Story* + short video industry consensus)

- **Source**: Storytelling (McKee's *Story*) + Short Video "3-Second Rule"
- **Core Idea**: Audience attention is driven by conflict, not statements. Your job is not to "tell" the audience what you're going to say, but to "pose" a problem they want to solve or a viewpoint they want to challenge.

**Theory Excerpt**:
> "Story is about conflict. No conflict, no drama. No drama, no audience attention." — Robert McKee

**How to Apply the Theory**:
The essence of a short video opening = Create cognitive conflict
- "You know X… but you don't know Y…" → Create a knowledge gap
- "Many people think X… actually Y…" → Create cognitive reversal
- "I once fell into this pitfall…" → Create relatable conflict

---

### Step 3: Process Specification

**Process Overview**: Topic Selection → Find Conflict Point → Write Conflict Opening → Validate → Expand

#### Step 1: Confirm Topic
- **Input**: The topic you want to talk about
- **Action**: Ask yourself "What is the biggest misconception the audience has about this topic?"
- **Output**: A "misconception statement"
- **Success Criteria**: Does this misconception statement resonate?

#### Step 2: Find the Conflict Point
- **Input**: Misconception statement
- **Action**: Rewrite it as a reversal statement
  - Misconception + Reversal = Conflict
- **Output**: "Many people think X, but actually Y" format
- **Success Criteria**: Is Y surprising but not absurd?

#### Step 3: Write the Conflict Opening Sentence
- **Input**: Reversal Y
- **Action**: Add a "counter-intuitive/data/pain point" hook before Y
- **Output**: "You know X… but actually Y…" opening
- **Success Criteria**: Within 5 seconds of reading, does it make people want to argue or listen further?

#### Step 4: Validation Test
- **Action**: Read it silently. Does it sound like you?
- **Exception Handling**: If not → Rewrite in conversational language; If too harsh → Add a scene for context

#### Step 5: Expand the Body
- **Action**: After the opening is confirmed, expand using the four-part structure: "Conflict → Clarify →干货 → Action"
