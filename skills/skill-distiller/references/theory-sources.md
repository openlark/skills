## How to Use

When the user describes a problem, first search this file for a corresponding theory. If found:
1. Tell the user the theory name and source
2. Ask if the theory fits their scenario
3. If it fits, use this theory for the "Theoretical Support" section

If not found, consider:
- Cross-domain transfer
- Original theory construction
- Combining multiple theories
