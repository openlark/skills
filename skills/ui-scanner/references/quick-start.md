# Quick Start

## 3-Step Scan

1. **Provide URL**: `Analyze the design system of https://stripe.com`
2. **Auto crawl**: web_fetch → CSS extraction (variables/style/inline) → 6-dimension classification
3. **Get output**: `{domain}_design.md` (e.g. `vercel.com_design.md`)

## Output Structure

YAML frontmatter(colors/typography/rounded/spacing/components) + design style + color system + typography + spacing & border radius + component styles + design guidelines(Do/Don't/breakpoints)
