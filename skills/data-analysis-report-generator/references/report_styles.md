# Report Style Reference

11 professional report styles for the HTML analysis report. Each defines color palette, typography, layout rules, and visual tone.

## Style Selection Guide

| Use Case | Recommended Style |
|---|---|
| Financial data, investment analysis | FT, Bloomberg, WSJ |
| Business strategy, consulting report | McKinsey, HBR |
| Macroeconomic, policy analysis | Economist |
| Scientific research, academic data | Nature, MIT |
| Tech industry, startup analytics | Wired, 36Kr |
| General news-style reporting | NYT |

## 1. FT (Financial Times)
- **Colors:** Salmon pink (#9E2B2E) on cream/white background
- **Typography:** Serif headings (Georgia-like), clean sans-serif body
- **Layout:** Single-column with generous whitespace, minimal grid lines
- **Tone:** Authoritative, understated, data-first
- **Chart style:** Clean bar/line charts, muted palette, pink accent

## 2. McKinsey
- **Colors:** Deep blue (#003A70), light blue (#0083CA), with grey accents
- **Typography:** Sans-serif (Arial/Helvetica), structured hierarchy
- **Layout:** Two-column, heavy structure, section dividers, executive summary box
- **Tone:** Structured, analytical, consulting-firm authoritative
- **Chart style:** Waterfall charts, structured bar charts, blue-dominant palette

## 3. Economist
- **Colors:** Classic red (#E3120B), dark navy (#0A2239), white
- **Typography:** Serif (ITC Officina or Economist-style), small text
- **Layout:** Multi-column newspaper layout, dense content, red accents
- **Tone:** Wry, intelligent, data-heavy, global perspective
- **Chart style:** Distinctive red-accent charts, clear data labels

## 4. Bloomberg
- **Colors:** Terminal green (#2ECC71), orange (#FF6B00), dark background (#1E1E1E)
- **Typography:** Monospace (Consolas/Courier), terminal aesthetic
- **Layout:** Dark theme, data-dense, sidebar panels, real-time feel
- **Tone:** Fast-paced, data-intensive, financial market focus
- **Chart style:** Dark-themed charts, candlestick, high-data-density

## 5. Harvard Business Review (HBR)
- **Colors:** Deep navy (#2C3E50), academic blue (#3498DB), muted red (#E74C3C)
- **Typography:** Serif (Merriweather-like) for body, sans-serif for headings
- **Layout:** Clean single-column, pull quotes, ample margin notes
- **Tone:** Academic yet accessible, research-backed, thoughtful
- **Chart style:** Professional academic charts, annotated insights

## 6. Nature
- **Colors:** Scientific blue (#2166AC), cool grey, warm red (#B2182B)
- **Typography:** Clean sans-serif (Helvetica), structured headings
- **Layout:** Multi-column scientific layout, abstract box, methods section
- **Tone:** Objective, precise, evidence-based, peer-review aesthetic
- **Chart style:** Scientific charts with error bars, statistical annotations

## 7. Wired
- **Colors:** Neon pink (#FF3366), electric blue (#00D2FF), dark backgrounds
- **Typography:** Bold sans-serif (Futura/Proxima Nova), large display fonts
- **Layout:** Magazine-style, large hero visuals, bold section breaks
- **Tone:** Forward-looking, energetic, tech-optimistic, bold
- **Chart style:** Bold, colorful, interactive, animated transitions

## 8. The New York Times (NYT)
- **Colors:** Classic black (#333333), grey (#999999), minimal color accents
- **Typography:** NYT Cheltenham (or Georgia fallback) headings, Imperial body
- **Layout:** Newspaper column layout, strong typographic hierarchy
- **Tone:** Trusted, comprehensive, explanatory journalism
- **Chart style:** Minimal, high-contrast, scrollytelling

## 9. Wall Street Journal (WSJ)
- **Colors:** WSJ navy (#0C2340), blue (#0068B4), institutional red (#A6192E)
- **Typography:** Serif (Escrow/Dow Jones style), traditional newspaper
- **Layout:** Dense columns, ticker-style data strips, traditional newspaper
- **Tone:** Business-authoritative, market-focused, institutional
- **Chart style:** Traditional financial charts, clear annotations

## 10. MIT Technology Review
- **Colors:** MIT red (#A31F34), charcoal (#5B6770), tech blue (#3388BB)
- **Typography:** Modern sans-serif (Neue Haas Grotesk), clean hierarchy
- **Layout:** Magazine-tech hybrid, innovation callouts, future-oriented
- **Tone:** Tech-deep, research-informed, innovation-focused
- **Chart style:** Modern data visualization, innovation-themed

## 11. 36Kr (36氪)
- **Colors:** Brand green (#00DAA7), deep space (#1A1A2E), cyber purple (#533483)
- **Typography:** Modern Chinese-friendly sans-serif (PingFang SC / Microsoft YaHei)
- **Layout:** Modern startup-media, card-based, industry tags, investment callouts
- **Tone:** Startup-ecosystem, trend-forward, data-driven tech journalism
- **Chart style:** Bold gradient charts, startup metrics focus
