# Emotion & Style Tags

Embed `[tag]` anywhere in the text, supports 15000+ free-form tags. Tags affect the following text until overridden.

## Common Tags

**Emotion:** `[excited]` `[angry]` `[sad]` `[surprised]` `[shocked]` `[delight]`

**Tone/Volume:** `[whisper]` `[low voice]` `[loud]` `[shouting]` `[screaming]` `[volume up/down]` `[echo]`

**Speaking Style:** `[emphasis]` `[with strong accent]` `[interrupting]` `[singing]`

**Breath/Pause:** `[pause]` `[short pause]` `[inhale]` `[exhale]` `[panting]` `[clearing throat]`

**Laugh/Sound Effects:** `[laughing]` `[laughing tone]` `[chuckle]` `[sigh]` `[tsk]` `[audience laughter]`

## Free-Form Examples

```
[whisper in small voice]This is a secret…
[professional broadcast tone]Good evening, dear listeners.
[pitch up]Higher pitch.
[angry but controlled]I don't want to see this again.
[excited and out of breath]I ran all the way here!
```

## Combinations

```
[excited]That's great![pause] [low voice]But keep it down…[laughing tone]Just kidding.
```

## Notes

- Free-form tags work best when described in English
- Place tags at the moment you want the tone to change, or slightly before
- Too many tags may cause instability
