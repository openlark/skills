# Startup Evaluation Report Template

## I. Project Overview
- Project name
- One-line description
- Current stage
- Core value proposition

## II. Market Analysis
### 2.1 Target Market
- Market size (TAM / SAM / SOM)
- Market growth trends
- Target user profile & demand validation

### 2.2 Competitive Analysis
- Key competitors and their respective strengths/weaknesses
- Project differentiation positioning
- Competitive moat assessment

### 2.3 Market Opportunity
- Market gaps & entry points
- Growth drivers

---

## III. Product/Service Assessment
### 3.1 Product Solution
- Core features & usage scenarios
- User experience & design
- Product maturity

### 3.2 Technical Innovation
- Core technology highlights
- Comparison with existing solutions
- Replicability & moat

---

## IV. Technical Feasibility
### 4.1 Technical Architecture
- Tech stack selection rationale
- Architecture scalability
- Development progress & milestones

### 4.2 Technical Risks
- Key technical challenges
- Third-party dependency risks
- Intellectual property status

---

## V. Business Model
### 5.1 Revenue Model
- Revenue sources & pricing strategy
- Unit economics (CAC / LTV / gross margin)
- Break-even estimate

### 5.2 Scaling Path
- Growth flywheel & acquisition channels
- Network effects / economies of scale
- Cost structure trends
