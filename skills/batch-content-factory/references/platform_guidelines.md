# Platform Content Guidelines Reference

## WeChat Official Accounts

### Content Characteristics
- Supports long-form, in-depth reading
- Strong text-image layout capability
- Suitable for brand building and deep content

### Best Practices
1. **Title**: 16-24 characters, spark curiosity
2. **Opening**: Capture attention in the first 3 seconds
3. **Structure**: Subheadings + body text + accompanying images
4. **Length**: 1500-3000 words optimal
5. **Engagement**: End with a discussion topic prompt

### Formatting Guidelines
```markdown
# Main Title

## Part 1
Body content...

![Image description](image.jpg)

## Part 2
Body content...

---

**Summary**
- Point 1
- Point 2

> Quoted content

[Read original article](link)
```

## Zhihu

### Content Characteristics
- Q&A format, highly professional
- High user quality, deep interaction
- Significant long-tail traffic

### Best Practices
1. **Answer Structure**: Viewpoint + Argument + Case study
2. **Professionalism**: Cite data and research
3. **Readability**: Use bold text and lists appropriately
4. **Engagement**: Reply to comments to increase exposure

### Formatting Guidelines
```markdown
## Core Viewpoint
One-sentence summary...

## Detailed Analysis
### 1. First Point
Arguments and cases...

### 2. Second Point
Arguments and cases...

## Summary
Review of key points...
```

## Xiaohongshu

### Content Characteristics
- Strong recommendation/product-seeding attributes
- Visually oriented
- Predominantly young users

### Best Practices
1. **Cover Image**: High aesthetic appeal, clear information
2. **Title**: Emoji + keywords
3. **Body**: Short sentences, clear segmentation
4. **Tags**: 3-5 relevant tags
5. **Engagement**: Guide likes and saves

### Formatting Guidelines
```
✨Title✨

First paragraph of body text...

💡Key Point 1
💡Key Point 2
💡Key Point 3

#Tag1 #Tag2 #Tag3
```

## Twitter / X

### Content Characteristics
- Strict character limit
- Strong real-time nature
- Rapid topic dissemination

### Best Practices
1. **Conciseness**: One viewpoint per tweet
2. **Topics**: Use relevant hashtags
3. **Engagement**: Ask questions to spark discussion
4. **Threads**: Post longer content using threads

### Formatting Guidelines
```
Core viewpoint (within 140 characters)

#Hashtag1 #Hashtag2

---

Thread format:
1/ Opening
2/ First point
3/ Second point
...
n/ Summary
```

## SEO Optimization Checklist

### Title Optimization
- [ ] Include core keywords
- [ ] Length 30-60 characters
- [ ] Trigger click-through desire
