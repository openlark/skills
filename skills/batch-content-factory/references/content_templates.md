# Content Creation Templates

## Tech Trend Article Template

### Structure
```
1. Phenomenon Introduction (trending event / data)
2. Trend Analysis (3-5 core trends)
3. Case Study Interpretation (2-3 typical cases)
4. Impact Assessment (impact on industry / individuals)
5. Actionable Advice (practical recommendations)
```

## Methodology Article Template

### Structure
```
1. Problem Definition (what problem it solves)
2. Methodology Introduction (core framework)
3. Step-by-Step Guide (specific operational process)
4. Case Demonstration (practical application)
5. Frequently Asked Questions (FAQ)
```

### Example
**Topic**: How to Build a Personal Knowledge Management System

## Story / Case Study Article Template

### Structure
```
1. Story Opening (engaging introduction)
2. Background Setup (character / environment introduction)
3. Conflict / Challenge (problems encountered)
4. Resolution Process (how it was overcome)
5. Outcome / Lessons Learned (gains and experience)
```

### Example
**Topic**: The Startup Story of an Indie Developer

---

## Checklist / Tool Article Template

### Structure
```
1. Introduction (why this checklist is needed)
2. Categorized List (organized by category)
   - Category 1
   - Category 2
   - Category 3
3. Usage Suggestions (how to use)
4. Update Notes (version information)
```

### Example
**Topic**: Essential Tools Checklist for Designers in 2024

---

## Opinion / Commentary Article Template

### Structure
```
1. Opinion Statement (clear stance)
2. Supporting Arguments (why you hold this view)
   - Argument 1
   - Argument 2
   - Argument 3
3. Rebuttal / Supplement (address possible counterarguments)
4. Summary and Elevation (reiterate core viewpoint)
```

### Example
**Topic**: Why I Believe AI Will Not Replace Programmers

---

## Quick Creation Checklist

### Pre-Publishing Check
- [ ] Is the title compelling?
- [ ] Does the opening capture attention?
- [ ] Is the structure clear?
- [ ] Are there any typos?
- [ ] Are the accompanying images appropriate?
- [ ] Is the CTA (Call to Action) clear?

### Platform Adaptation Check
- [ ] Does the word count meet platform requirements?
- [ ] Does the formatting comply with platform standards?
- [ ] Are tags / hashtags added?
- [ ] Is the publishing time appropriate?
