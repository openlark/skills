# Thinking Models Reference Index

This document provides a quick reference for all thinking models needed for deep article analysis.

## Model Navigation

| Thinking Model | Applicable Scenarios | Core Question |
|----------------|---------------------|---------------|
| [SCQA Framework](scqa.md) | Grasp the logical thread of the article | Situation → Complication → Question → Answer |
| [5W2H Analysis](5w2h.md) | Comprehensive information scanning | What / Why / Who / When / Where / How / How much |
| [First Principles](first-principles.md) | Trace back to the source | What is the most fundamental truth? |
| [Six Thinking Hats](six-thinking-hats.md) | All-around evaluation | What does each perspective see? |
| [Critical Thinking](critical-thinking.md) | Argument assessment | Is the evidence sufficient? Is the logic valid? |
| [Systems Thinking](systems-thinking.md) | Complex system analysis | Interconnections and feedback among elements? |
| [Reverse Thinking](reverse-thinking.md) | Identify weaknesses | What if we think in reverse? |
| [Mental Models](mental-models.md) | Framework examination | Which cognitive framework to use for understanding? |
| [Comparison Matrix](comparison-matrix.md) | Option comparison | What are the pros and cons of each option? |
| [Inversion Thinking](inversion-thinking.md) | Risk prevention | How can we ensure we don't fail? |

## Usage Recommendations

### Beginner's Path
1. First, scan basic information using 5W2H
2. Grasp the logical structure using SCQA
3. Examine arguments using Critical Thinking
4. Organize viewpoints using the Comparison Matrix

### Advanced Combinations
- **In-Depth Analysis**: First Principles + Systems Thinking + Critical Thinking
- **Decision Reference**: Six Thinking Hats + Comparison Matrix + Inversion Thinking
- **Investment / Business**: Mental Models + Systems Thinking + Reverse Thinking
- **Risk Assessment**: Inversion Thinking + Reverse Thinking + Critical Thinking

### Selection by Article Type
- **News Reports**: 5W2H + Critical Thinking + Comparison Matrix
- **Business Analysis**: SCQA + Systems Thinking + Mental Models
- **Op-Eds / Commentary**: Critical Thinking + Reverse Thinking + Six Thinking Hats
- **Academic Papers**: First Principles + Critical Thinking + Systems Thinking
- **Strategic Recommendations**: SCQA + Comparison Matrix + Inversion Thinking
