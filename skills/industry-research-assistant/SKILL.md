# General Industry Research Assistant

Conduct structured analysis of any industry, select appropriate framework combinations, and output research reports.

## Use Cases

Use when users need "industry research", "industry analysis", "market research", "competitive analysis", or "industry report".

## Workflow

### 1. Define Research Objectives

Confirm with the user:
- Target industry and sub-sector
- Research purpose (investment decision / market entry / competitive analysis / strategic planning)
- Geographic scope (global / specific country or region)
- Time dimension of interest (current snapshot / trend forecast / historical review)

### 2. Select Analytical Frameworks

Based on research purpose, combine from the following frameworks (see [references/frameworks.md](references/frameworks.md) for details):

| Framework | Applicable Scenario |
|-----------|-------------------|
| **PESTEL** | Macro-environment scan (Political, Economic, Social, Technological, Environmental, Legal) |
| **Porter's Five Forces** | Industry competitive structure analysis |
| **SCP** | Structure-Conduct-Performance: analyze industry landscape & efficiency |
| **SWOT** | Enterprise/industry strengths, weaknesses, opportunities & threats |
| **Supply-Demand Analysis** | Supply-demand balance, elasticity, driving factors |
| **Cycle Assessment** | Industry lifecycle, economic cycle positioning |

For framework usage guidelines, analysis dimensions, and output points, see [references/frameworks.md](references/frameworks.md).

### 3. Key Indicator Extraction

Extract corresponding core indicators for different industry types (see [references/key-indicators.md](references/key-indicators.md) for details):

- Market size & growth rate (CAGR)
- Concentration (CR4/CR8/HHI)
- Profit margin levels & trends
- Entry/exit barriers
- Capacity utilization
- Price elasticity
- Policy sensitivity

### 4. Report Output

Output a structured research report. Recommended structure:

```
I. Industry Overview (definition, classification, scale, development stage)
II. Macro-Environment Analysis (PESTEL)
III. Competitive Landscape Analysis (Porter's Five Forces + SCP)
IV. Supply-Demand Analysis (supply-demand structure, drivers, pricing mechanisms)
V. Cycle Assessment (lifecycle stage, economic cycle positioning)
VI. SWOT Summary
VII. Key Data Summary
VIII. Conclusions & Recommendations
```

Adjust sections based on research purpose and industry characteristics — not all sections need to be covered.
